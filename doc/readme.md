## 一个游戏设计稿

初步想法，做一个扑克牌游戏。

### 核心机制

- 两个人都有11张手牌，共比赛11轮。
- 两个人各有 $x$ 张牌是明牌，剩下的牌是暗牌。
- 两个人轮流出牌，每次选择手牌中任意一张。
- 每一轮两张牌比点数大小，点数大的一方获胜（同点数比花色，黑(0.9)红(0.6)梅(0.4)方(0.1)），出过的牌都遗弃。
- 最值比较获胜次数，获胜次数多的一方获胜，获胜次数差值**越小**得分越高。
